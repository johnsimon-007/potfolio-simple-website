# Personal Portfolio Website

## Description

This is a simple personal portfolio website built using **HTML and CSS**.
The website includes sections such as **About Me, Interests, Projects, and a Contact Form**.

This project was created to practice **semantic HTML structure, basic CSS styling, and website layout design**.

---

## Features

* About Me section
* Interests section
* Projects showcase
* Contact form page
* Social media link

---

## Technologies Used

* HTML5
* CSS3
* Google Fonts

## Purpose of the Project

The goal of this project is to practice:
